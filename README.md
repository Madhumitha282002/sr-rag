# SR-RAG: Super-Resolution Research Paper Assistant

A RAG system for searching, comparing, and citing image super-resolution research papers.

## Tech Stack
- **Embeddings**: sentence-transformers
- **Vector DB**: ChromaDB
- **LLM**: OpenAI / Gemini / Ollama
- **Frontend**: Streamlit
- **API**: FastAPI
